Gerando pdfs com xelatex

No arquivo .Rnw foram inseridos non-printable unicode characters obtidos aqui.

Para executar um container baseado na imagem docker da ploa 2024 abra o programa Docker Desktop e depois execute no terminal:

docker run --rm -p 8787:8787 -ti --mount type=bind,source=$(PWD),target=/home/rstudio volumes:ploa2024 bash

Dentro do container execute:

R CMD Sweave --pdf table.Rnw 
pdfinfo table.pdf
